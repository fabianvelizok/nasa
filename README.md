## Directory Structure

* **server/**: Contains the Node.js server code.
* **client/**: Contains the Create React App code.

## Running the Application
Install dependencies:
```
npm run install
```

Start development server:
```
npm run dev
```
Deploy production:
```
npm run deploy
```
Deploy production cluster:
```
npm run deploy:cluster
```
